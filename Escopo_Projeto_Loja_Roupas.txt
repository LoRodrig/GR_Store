Projeto: Sistema de Registro de Vendas de Roupas

Objetivo

Desenvolver um sistema web simples para gerenciamento de uma loja de
roupas, permitindo cadastrar clientes, vendedores, produtos e registrar
vendas.

Tecnologias: - React + Vite - Tailwind CSS - React Router - Lucide
React - Supabase PostgreSQL - GitHub Pages

Funcionalidades

-   Dashboard
-   CRUD de Clientes
-   CRUD de Vendedores
-   CRUD de Categorias
-   CRUD de Produtos
-   Registro de Vendas
-   Histórico de Vendas

Estrutura do Banco

Tabelas: - clientes - vendedores - categorias - produtos - vendas -
venda_itens

O que já foi feito

-   Projeto criado no Supabase.
-   Banco de dados criado.
-   Tabelas criadas.
-   Relacionamentos (FKs) configurados.
-   Índices criados.
-   Categorias iniciais inseridas.
-   RLS habilitado.
-   Policies criadas.
-   Identificado Project URL e Publishable Key para integração.
